Fill Mode

Privacy Policy / 개인정보처리방침

Last updated: 2026-07-02

English

Fill Mode does not collect, store, transmit, or sell any personal data.

  • No data collection. The extension has no analytics, no tracking, and no external server. It never reads page content other than locating <video> elements to apply a visual CSS transform.
  • Local settings only. Your preferences (aspect-ratio presets, zoom level, keyboard shortcut mapping, options) are stored with the Chrome storage API: chrome.storage.sync (synced to your own Google account by Chrome) and chrome.storage.session (cleared when the browser closes). This data never leaves your browser except through Chrome's own settings sync.
  • No remote code. All code ships inside the extension package.

If you have questions, please open an issue on the project repository.

한국어

Fill Mode는 어떠한 개인정보도 수집·저장·전송·판매하지 않습니다.

  • 데이터 수집 없음. 분석 도구, 트래킹, 외부 서버가 없습니다. 페이지에서 <video> 요소를 찾아 CSS 시각 변형을 적용하는 것 외에 페이지 내용을 읽지 않습니다.
  • 설정은 로컬에만 저장. 사용자 설정(비율 프리셋, 배율, 단축키 매핑, 옵션)은 Chrome storage API에 저장됩니다: chrome.storage.sync(Chrome이 사용자 본인의 Google 계정으로 동기화)와 chrome.storage.session(브라우저 종료 시 삭제). 이 데이터는 Chrome 자체의 설정 동기화 외에는 브라우저 밖으로 나가지 않습니다.
  • 원격 코드 없음. 모든 코드는 확장 프로그램 패키지 안에 포함되어 있습니다.

문의는 프로젝트 저장소의 이슈로 남겨주세요.
