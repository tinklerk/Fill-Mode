비디오의 레터박스(위아래·양옆 검은 여백)를 제거하는 확장 프로그램입니다. 화면 비율과 영상 비율이 다를 때 생기는 검은 띠를 없애고, 영상이 화면을 꽉 채우도록 확대·크롭합니다.

기존의 비슷한 확장들과 달리, Fill Mode는 세로(피벗) 모니터와 사용자 정의 비율을 제대로 지원합니다. 세로 모니터에서 16:9 영상을 볼 때, 초광각 모니터에서 일반 영상을 볼 때 특히 유용합니다.

■ 주요 기능

• 비율 프리셋 10종 — 가로(16:10, 16:9, 18:9, 21:9, 32:9)와 세로(9:16, 9:18, 9:21, 10:16, 3:4)
• 커스텀 비율 — "9:19.5" 같은 비율이나 "2160x3840" 같은 해상도를 직접 입력
• 수동 배율 조절 — 0.05 단위로 0.5~4.0배까지 미세 조정
• 위치 이동(Pan) — 확대로 잘린 부분을 D-pad나 숫자 입력으로 이동. 위치는 사이트별로 기억되어 다음 영상에서도 유지되고, 위치만 따로 초기화할 수 있습니다
• 전체화면 지원 — 페이지 내 단축키로 전체화면에서도 조작하고, 결과는 화면 위 HUD로 즉시 확인
• 단축키 변경 — 모든 단축키를 설정 페이지에서 원하는 키로 변경. 한글 자판 상태에서도 동작합니다
• 6개 언어 지원 — 한국어, 영어, 일본어, 중국어 간체, 스페인어, 프랑스어

■ 사용 방법

1. 영상이 있는 페이지에서 툴바의 Fill Mode 아이콘을 클릭합니다
2. 팝업에 모니터와 영상의 해상도·비율이 표시됩니다
3. 내 모니터와 같은 비율의 프리셋을 누르면 레터박스가 사라집니다
4. 필요하면 +/− 버튼으로 배율을, D-pad로 보이는 위치를 조정합니다

■ 기본 단축키 (페이지 내, 전체화면에서도 동작)

• Z — 켜기/끄기
• X — 비율 프리셋 순환
• [ / ] — 배율 감소/증가
• Shift + 화살표 — 위치 이동
• Shift + X — 위치 초기화

■ 지원 사이트

YouTube, Netflix, Disney+는 플레이어 구조에 맞춰 최적화되어 있고, 그 외 사이트에서도 페이지에서 가장 큰 영상을 자동으로 찾아 동작합니다. iframe 안의 플레이어도 지원합니다.

■ 개인정보

Fill Mode는 어떤 데이터도 수집·전송하지 않습니다. 분석 도구도 외부 서버도 없으며, 설정은 브라우저의 저장소(chrome.storage)에만 저장됩니다. 모든 사이트 접근 권한은 사용자가 방문하는 비디오 사이트에서 영상을 찾아 CSS 확대를 적용하기 위한 것으로, 페이지 내용을 읽거나 수집하지 않습니다.
