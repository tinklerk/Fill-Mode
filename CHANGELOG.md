# 변경 이력

이 프로젝트의 모든 주요 변경사항은 이 파일에 기록됩니다.

형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.0.0/)를 따르며,
이 프로젝트는 [Semantic Versioning](https://semver.org/lang/ko/)을 준수합니다.

이 저장소는 Netflix, YouTube, Disney+ 등의 비디오에서 레터박스(검은 여백)를 제거하는
크롬 확장 프로그램(Fill Mode)입니다. `<video>` 엘리먼트에 CSS `transform: scale()`을
적용해 화면 비율에 맞게 확대/크롭하며, 기존 Zoom to Fill 계열 확장과 달리 세로(portrait)
모니터 프리셋과 사용자 정의 비율 입력을 지원합니다. 1.0.0은 프리셋·커스텀 비율·수동
배율·전역 단축키를 갖춘 첫 정식 릴리스를 뜻합니다.

## 2.0.0 - 2026-07-02

다국어 지원과 위치 이동(pan), 전체화면 조작을 더한 두 번째 메이저 릴리스입니다. 팝업·옵션
UI를 전면 재디자인하고 아이콘을 교체했으며, 페이지 내 단축키와 HUD로 전체화면에서도
확장을 조작할 수 있습니다. 단축키는 옵션 페이지에서 원하는 키로 바꿀 수 있고 한글 자판
상태에서도 동작합니다.

### 추가됨 2.0.0

- **다국어 지원(6개 언어)** — `chrome.i18n` 기반으로 한국어·영어·일본어·중국어 간체·
  스페인어·프랑스어를 지원. 확장 이름/설명, 전역 단축키 설명, 팝업·옵션의 모든 UI
  문자열을 `_locales/` 메시지로 관리 (기본 언어: 영어)
- **위치 이동(pan)** — 확대로 잘린 크롭 중심을 D-pad(길게 누르면 연속 이동)와 X/Y 숫자
  입력(-50~50%)으로 이동. 위치는 사이트별로 세션에 보존되어 다음 비디오나 새로고침에도
  유지되고, 위치만 따로 초기화 가능
- **페이지 내 단축키 + HUD** — `z` 토글, `x` 프리셋 순환, `[` `]` 배율 조절,
  `Shift+화살표` 위치 이동, `Shift+X` 위치 초기화. 전체화면에서도 동작하며 조작 결과를
  화면 상단 HUD 토스트로 표시. 입력창 포커스 중이거나 페이지에 video가 없으면 반응하지
  않음
- **단축키 변경** — 옵션 페이지에서 각 단축키를 클릭 후 키 입력으로 리맵
  (`settings.keymap`). 열려 있는 탭에 새로고침 없이 즉시 반영되고, 팝업에도 현재 키맵을
  보여주는 접이식 단축키 안내를 추가

### 변경됨 2.0.0

- **팝업 UI 재디자인** — 토글 스위치, 정보 카드, 알약형 프리셋 칩(활성 시 그라디언트
  강조), 원형 스테퍼로 전면 개편
- **옵션 페이지 재디자인** — 라디오 버튼 나열을 세그먼트 컨트롤 카드로 바꾸고, 시스템
  라이트/다크 테마에 자동 대응
- **아이콘 교체** — 둥근 모서리의 인디고 그라디언트 배경에 캡슐(pill) 모티프,
  안티앨리어싱과 투명 코너 적용
- **비율 표시 형식** — 모니터·비디오 정보를 `1080x1920 (9:16)`처럼 해상도 + W:H 비율로
  표시해 프리셋 선택과 비교하기 쉽게 변경 (약분이 안 되는 해상도는 `≈16:9`처럼 근사)
- **기본 전역 단축키 제거** — 배율 증가/감소 커맨드의 `suggested_key`를 없애 새 설치에서
  "설정 안 됨"이 기본이 되도록 변경

### 수정됨 2.0.0

- 한글(IME) 자판 상태에서 페이지 내 단축키가 동작하지 않던 문제를 수정 — `e.key` 대신
  물리 키 코드(`e.code`)를 정규화해 자판 언어와 무관하게 동작

## 1.0.0 - 2026-07-02

Manifest V3 기반의 첫 정식 릴리스입니다. 가로·세로 프리셋과 커스텀 비율 입력, 수동 배율
조절, 전역 단축키를 갖췄고 YouTube·Netflix·Disney+의 플레이어 구조에 맞춘 비디오 탐지를
제공합니다.

### 추가됨 1.0.0

- **비율 프리셋 10종** — 가로(16:10, 16:9, 18:9, 21:9, 32:9)와 세로(9:16, 9:18, 9:21,
  10:16, 3:4) 프리셋을 팝업에서 2열로 제공. 세로 프리셋은 피벗/세로 모니터 사용자를 위한
  이 확장의 차별점
- **커스텀 비율 입력** — `W:H`(예: `9:19.5`) 또는 해상도 `WxH`(예: `2160x3840`) 형식을
  파싱해 임의 비율로 적용. 잘못된 형식은 입력창에서 거부
- **수동 배율 스테퍼** — 팝업의 `− x1.00 +` 스테퍼로 배율을 0.05 단위, 0.5~4.0 범위에서
  직접 조절
- **전역 단축키 4종** — 배율 증가/감소(`Ctrl/Cmd+↑`·`Ctrl/Cmd+↓`), 프리셋 순환, 초기화.
  백그라운드 서비스 워커가 활성 탭의 content script로 커맨드를 라우팅
- **사이트별 비디오 탐지** — YouTube·Netflix·Disney+는 알려진 셀렉터로, 그 외 사이트는
  화면에서 가장 큰(재생 중 가중치) `video`를 폴백으로 선택. iframe 내부 플레이어를 위해
  모든 프레임에 주입
- **SPA 네비게이션 대응** — MutationObserver와 YouTube의 `yt-navigate-finish` 이벤트로
  페이지 이동 없이 교체되는 video를 감지해 상태를 재적용. 창 리사이즈·전체화면 전환 시에도
  배율 재계산
- **크롭 컨테이너 보장** — 확대 시 넘친 영역이 잘리도록, 플레이어 컨테이너에
  `overflow: hidden`이 없으면 클래스를 주입해 보장
- **팝업 UI** — On/Off 토글, 모니터·비디오 비율 표시, 배율 스테퍼, 프리셋 버튼,
  커스텀 입력, 초기화 버튼으로 구성된 다크 테마 팝업
- **옵션 페이지** — Persistent zoom mode(Off/Session/Permanent), 줌 전환 속도
  (0/150/300/500ms), 모니터 비율 소스(Screen/Window) 설정을 `chrome.storage.sync`에 저장
